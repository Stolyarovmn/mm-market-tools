# Проверка top-10 продавцов категории

Дата: 2026-04-08

Цель: проверить, являются ли продавцы из ранее собранного top-10 именно сильными продавцами внутри категории `Игрушки и игры (10162)`, или часть из них это широкие магазины с небольшой долей игрушек.

## Главное уточнение по старому рейтингу

Ранее top-10 строился по полю `products_seen` из полного обхода категории.

Это полезный сигнал, но он не равен строгому числу уникальных SKU в категории. В обходе могли накапливаться:

- повторные попадания товара в разных листовых подкатегориях;
- дубли по вариациям;
- повторные показы одного и того же ассортимента в разных сегментах выборки.

То есть старый top-10 нужно читать как:

- "кто больше всего встречался в категории в полном обходе",

а не как:

- "строгое место по уникальным SKU в игрушках".

## Независимая проверка через публичный поиск MM

Для каждого продавца из старого top-10 были сняты два числа:

- `all`: весь публичный ассортимент магазина;
- `toys`: публичный ассортимент только внутри категории `10162`.

## Вывод по теории

Теория подтвердилась частично.

### Кто реально игрушечный продавец

- `LEGO`
- `BUDI BASA`
- `Академия Игр`
- `Ultra Toys`
- `Кунай`
- `OYLAA!`
- `Удачная Тема`

У этих продавцов игрушки занимают основную или очень значимую часть ассортимента.

### Кто широкий магазин, а не чисто игрушечный

- `FBS ООО «Джой Групп» FBS`
- `Sima-Express`
- `ExclusiveGOODS`

Особенно показателен `Джой Групп`: почти `211 тыс.` товаров в публичном ассортименте и только `998` в игрушках. Это очень широкий магазин, а не специализированный игрок категории.

## Что видно по первым товарам

По выборке первых публичных карточек внутри категории:

- `Кунай` выглядит как специализированный ассортимент по игровым ножам, лукам, деревянному оружию и подростковым игрушкам;
- `Академия Игр` выглядит как широкий, но всё же игрушечный магазин: куклы, машинки, конструкторы, мягкие игрушки;
- `LEGO` и `BUDI BASA` это очевидные категорийные специалисты;
- `Sima-Express` это широкий маркет-магазин, у которого игрушки только одна из больших витрин;
- `Джой Групп` внутри игрушек представлен, но сам магазин явно не игрушечный.

## Практический вывод для аналитики

1. Старый top-10 нельзя считать "десятью главными прямыми игрушечными конкурентами" без уточнения метрики.
2. Для вашей конкурентной аналитики правильнее делить их на:
   - категорийных специалистов;
   - широких продавцов с заметным игрушечным разделом.
3. Для роста вашего магазина полезнее ориентироваться в первую очередь на:
   - `Академия Игр`
   - `Ultra Toys`
   - `LEGO`
   - `BUDI BASA`
   - `Кунай`

И отдельно держать в уме широких игроков как фон рынка:

- `Джой Групп`
- `Sima-Express`
- `ExclusiveGOODS`
